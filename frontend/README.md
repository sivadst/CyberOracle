# HOME.AI — Neural Intelligence Command Center

![HOME.AI Hero](https://images.unsplash.com/photo-1550751827-4bd374c3f58b?auto=format&fit=crop&q=80&w=2070)

The epicenter of next-generation AI architecture. Powering the future of telemetry, cybernetics, and autonomous decision ecosystems. 
This is the production-grade front-end for the **HOME.AI** ecosystem, integrating real-time telemetry from **CyberOracle**, predictive mapping from **NASA NEO**, and more.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS V4 & PostCSS
- **Animations**: Framer Motion
- **Data Visualization**: Recharts, D3.js, Cobe (3D Globe)
- **State Management**: Zustand
- **Icons**: Lucide React

## Ecosystem Matrix

1. **CyberOracle** - Predictive threat intelligence matrix.
2. **LifeLens AI** - Biometric and cognitive state analysis.
3. **SmartCommuteX** - Autonomous grid optimization.
4. **NASA NEO** - Near-Earth Object trajectory prediction.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the command center.

## Production Build

```bash
npm run build
npm run start
```

## Features
- **Cinematic Intro**: Boot sequence and terminal-style loader.
- **Glassmorphism**: Advanced transparent UI layers.
- **Live Telemetry**: Real-time graphs simulating backend websocket data.
- **Interactive 3D Globe**: Built with Cobe for spatial visualization.
